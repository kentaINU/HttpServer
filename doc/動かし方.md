# MyWebSystem 実行ガイド

このプロジェクトは、C# (.NET 8.0) で実装された自作 HTTP サーバーです。
ネットワーク処理（App）、解析ロジック（Core）、自動テスト（Tests）の3プロジェクト構成になっています。

---

## 🚀 クイックスタート (VS Code Task)

VS Code のタスク機能を使うと、コマンドを打たずに操作できます。

1.  **コマンドパレットを開く**: `Cmd + Shift + P` (Mac)
2.  **タスクの実行**: `Tasks: Run Task` を選択
3.  **サーバー起動**: `1. Server: Run (App)` をクリック
    * ※ `Cmd + Shift + B` でも実行できるように設定されています。

---
